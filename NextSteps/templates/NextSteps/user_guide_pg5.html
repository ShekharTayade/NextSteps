Search Features


With free or subscribed access you will be able to search Insitutes, Entrance Exams,Programs (Courses), Survey Ranks, JEE Opening/Closing Ranks
Institutes Search Colleges/Institutes by states and cities, name or institute type
  • From home page, click on “Location” dropdown to select states and cities to search in
  • You can enter the name if you want to search for a particular institute
  • Click on the magnifying glass button to obtain the list of institutes
  • Search the institutes that you are interested. Use the filters on the left to narrow the institutes search
  • Click the checkbox against the institute to select it for comparison
  • Select up to 4 institutes for comparison
  • As you select more than one institutes, a “Compare” button appear. Click it to get the results
  • Clicking on “Add to Preferences” will add that institute in your preferences
  • You can further narrow the search result using the “Search…” box at the top of the page. This is a free text search that will search in name, address or any information displayed for the institute.

Exams Get a list of institutes that have the selected entrance exams as admission criteria
  • On the home page search section, click on the “Exams” tab
  • Use the “Exam Level” dropdown to select the level of exam such as national, state, university or institute level exam
  • You can also enter the exam name/acronym if you want to search for a particular institute
  • Click on magnifying button icon to obtain the list of exams
  • Click on the entrance exam name/acronym to get the list of institutes that have the given entrance exam as criteria for admissions.
  • You can further narrow the search result using the “Search…” box at the top of the page. This is a free text search that will search in any information displayed.

Programs Get a list of programs and the institutes that offer those
  • On the home page search section, click on the “Programs” tab and click then on the “Programs” dropdown to select the program you are interested in
  • Enter the name of the institute if you are interested in a particular institute
  • Click on the magnifying glass button
  • A list of institutes offering those programs is displayed
  • You can further narrow the search result using the “Search…” box at the top of the page. This is a free text search that will search in any information displayed.
  • You can export this data for your reference in various formats such as Excel, PDF or print it on the connected printer.

Survey Ranking Find the ranking of institutes as published by National Institutional Ranking Framework (NIRF).
  • On the home page search section, click on the “Survey Rank” tab
  • Enter the name of the institute if you are interested in a particular institute
  • Click on the magnifying glass button
  • A list of institutes and their ranking is displayed
  • You can further narrow the search result using the “Search…” box at the top of the page. This is a free text search that will search in any information displayed.
  • You can export this data for your reference in various formats such as Excel, PDF or print it on the connected printer.

JEE Opening/Closing Ranking Obtain the opening and closing ranks for various programs and institutes.
  • On the home page search section, click on the “JEE Ranks” tab
  • Select rank type from (ADV or MAIN) the from Rank Type dropdown and select the year from “Year” dropdown
  • Click on the magnifying glass button obtain the list of institutes, programs and opening and closing ranks
  • You can further narrow the search result using the “Search…” box at the top of the page. This is a free text search that will search in any information displayed. For an example, if you want search only for the Mechanical Engineering opening, closing ranks, then enter “Mechanical Engineering” in the “Search Program” search box.
  • You can export this data for your reference in various formats such as Excel, PDF or print it on the connected printer.
