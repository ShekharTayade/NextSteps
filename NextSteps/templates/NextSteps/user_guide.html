NextSteps.co.in - User Guide

NextSteps has a database of over 6500 engineering colleges across India. When you Sign Up for free you will get access to search the colleges/Institutes by location, type of college, get details such as programs/courses offered, seats and quotas, institute survey ranks, cut offs for various programs, and JEE opening and closing ranks etc. When you Sign Up and Register + Subscribe you will have access to the powerful NextSteps features that will save you time and effort you would otherwise spend in doing it manually. It tracks your preferred/shortlisted programs, institutes and provide with all the information you need as you will go through the application, entrance exam, results processes. For JEE, it will also give you the seat probability based on your expected rank range and previous year opening, closing ranks.

Free and Paid Access

You can just sign up and use NextSteps for absolutely free. It lets you search Institutes/Colleges by programs/courses, seats and quotas, location, type of Institute, survey ranking, opening & closing JEE ranks,  and cut offs for various Institutes. This an important information as you research colleges that are best fit for you. All the free access features are in SEARCH menu.

However, the real power of NextSteps is when you sign up and pay a nominal fee and make use of the NextSteps as your real digital assistant. It will give you an access to productivity features that save a lot of time and effort so you concentrate on your studies. These features give you access to PREFERENCES and MY SPACE menu.

Common Usage Norms

When a page displays a list of items, you may want to select multiple values. For an example, a list displaying programs/courses where you may be interested in more than one program/course and hence would want to select multiple values. In such cases while on desktop/laptop you can use ‘ctrl + click’ to select multiple values, and while on mobile devices you would get a popup with checkboxes that will let you select multiple values.

As you choose values and get your filter criteria ready, you would typically click a “Get Xxxxxx” button to obtain the result. Depending on the context the ‘Xxxxxx” could “Institutes”, “Details” etc.

After you made selections on the pages that lets you search/find, obtain the details, you would click on the “Get Xxxxxx” button to get the details. The result page would have pagination in case the results span multiple pages.

The results pages would typically have a couple additional filters to narrow the results further. Please note that if there is a pagination, these filters work only on results on the current page.

 

SEARCH Menu

With free or paid access you will be able to use the SEARCH menu. It has following options.

Institutes by Location     Search Colleges/Institutes by states and cities.

  • There are two tabs, State and City.
  • Select the states that you want to search the colleges in. Don’t make any selection if you want to use all the states to search.
  • Move to City tab by clicking it. It will display the colleges in the state(s) you selected. Select the cities you want to search colleges in. Don’t make any selection if you want to use all the cities to search.
  • After you are done making the state and city selections, click on the “Get Institutes” button.

Survey Ranking                Find the ranking of institutes as published by National Institutional Ranking Framework (NIRF).

  • To select the year you are interested in click on the toggle stich next to it. The List of institutes with their ranks will be filtered for the selected year.
  • Use the slider to select the ‘from’ and ‘to’ ranks. The List of institutes with their ranks will be filtered for the ranks between selected to and from ranks.

Institutes by Type            Search Colleges/Institutes by the type.

  • Select the institute type that you want to search the colleges for by clicking on the toggle switch. Don’t make any selection if you want to use all the types.
  • Click on the “Get Institutes” button at the bottom to obtain the list of institutes.

Opening/Closing JEE Ranks         Obtain the opening and closing ranks for various programs and institutes.

  • Select the years and the type of institutes that you want to search the ranks for by clicking on the toggle switches. Don’t make any selection if you want to use all the values.
  • Click on the “Get Details” button at the bottom to obtain the list of institutes, programs and ranks.

Cut off Scores                   Obtain the cut off scores for various institutes and programs

  • Select the years that you want to search the cut off scores for by clicking on the toggle switches. Don’t make any selection if you want to use all the values.
  • Click on the “Get Details” button at the bottom to obtain the list of institutes, programs and ranks.

Programs                           Get the list of institutes offering the selected programs.

  • Select the programs that you want to search the colleges in. Don’t make any selection if you want to use all the programs to search.
  • To select multiple values use ctrl + click.
  • After you are done making selections, click on the “Get Institutes” button.

Seat Categories and Quotas        find the seat quotas by student categories.

  • Select the student category that you want to find the seats for. Don’t make any selection if you want to use all the categories.
  • Move to Programs tab by clicking it. It will display the list of Programs. Select the programs that you want to search seats for. Don’t make any selection if you want to use all the programs.
  • After you are done making the selections, click on the “Get Institutes” button.

 

PREFERENCES Menu

With registration and active subscription you will be able to use the PREFERENCES menu. It has following options.

View/Set My Preferences            View and Set the programs, institutes that you want to shortlist/target for admission/have preference for.

  • The preferences you set here will be the basis for the details under "MY SPACE" menu
  • Please use the ""Set Your Preferences" section to SET the preferences
    • Use the "Student Category" tab to select your category. This will be used while generating the reports for you.
    • Use the "Programs" tab to select the programs you are interested in
    • As you move to "Institutes" tab it will display the Institutes that offer those programs
    • Select the Institutes that you want to set as your preference
    • Use Control + click to select multiple choices
    • After you have made the selections please click the "Set Preferences" button to save the preferences
  • Your currently saved preferences are shown in "Your Current Preferences" section below
    • Click on + or - icon to expand or collapse
    • Your currently saved preferences are displayed under respective areas

My College Application Details   Save the details you would typically need filling up the application forms for various institutions.

  • Use the left side navigation bar to go to respective set of information to save.
  • Please note that only name mandatory. You can choose whatever other information you want to save.
  • There are sour sets of information – Name and Personal Details, Address and Contact Details, Education Details, Parents & Guardian, Photo and Signatures
  • The images to be uploaded in Photo and Signatures section should be <=50 KB, as most of the institutions won’t allow images of size larger than that.

 

MY SPACE Menu

With registration and active subscription you will be able to use the MY SPACE menu. The options under this menu work on the programs and institutes that you have set in your preferences (PREFERENCES Menu -> View/Set My Preferences). It has following options.

Institute Details Clicking on this option displays a list of your preferred institutes and programs along with details of seats and quotas.

Important Dates              Clicking on this option displays the important dates for your preferred institutes. Important dates such as application form start date, close date, entrance exam date, results date, etc.

Admission Routes            Clicking on this option displays the various ways that you can explore to get admission in your preferred institutes.

Entrance Exam                 Clicking on this option displays the details on the entrance exams for your preferred institutes.

Consolidated Report       Clicking on this option displays all the information discussed so far for your preferred institutes. This information will be in a PDF format that you can print out or save.

Calendar                            Clicking on this option displays a calendar of all the important dates that you need to be aware for your preferred institutes.

 

Institute, Program Ranks              Display the JEE opening & closing ranks for your preferred or selected institutes and programs.

  • Select whether you want to use your preferences to display the ranks.
  • Select 'Yes' if you would like to use the preferences that you have saved (PREFERENCES Menu -> View/Set My Preferences). JEE ranks will be displayed only for those programs and Institutes.
    You can still choose to select programs and institutes within those preferences. Please use 'ctrl + click' to select multiple program, institutes.
  • If you choose ‘No’, then list of all the programs and institutions will be displayed and there will be an additional selection box.
  • In the additional selection box, select 'Yes' if you would like to view all the Programs and Institutes and then select the ones you would like to be find the ranks for. If you select ‘No’ then all the programs and institutes will be used.
  • Select whether you would like to see ranks for JEE MAIN or JEE ADVANCED. The list of programs and institutes will be filtered accordingly.
  • After you are done with your selections, click on “Get Ranks” button at the bottom to obtains opening and closing ranks for your selections.

Seat Probability Display the list of institutes and programs where you have changes to get a seat.

  • The first selection is to select whether you want to use your preferences to display the ranks.
  • Select 'Yes' if you would like to use the preferences that you have saved (PREFERENCES Menu -> View/Set My Preferences). These Programs and Institutes will be used in Seat Probability analysis. If you select ‘No’, then the list of all the programs and institutes will be displayed and an additional selection box “Select Programs, Institutes” will be displayed.
  • The second selection box is to select whether you would like select any programs and institutes from the displayed lists. If you select ‘No’ then the list of programs and institutes will become removed from the page and all the programs and institute will be used.
  • From ‘Get Results By” selection box, select whether you want to use JEE MAINS and JEE ADV. If you choose MAINS, then the ‘Home State’ selection box will be enabled and the programs and institutes will be filtered for MAINS. If you choose ADV, then the ‘Home State’ selection box will be disabled and the programs and institutes will be filtered for ADV.
  • If the program and institutes selection boxes are visible then select programs and institutes that you want to choose for assessing the seat chances.
  • Next, enter the JEE expected rank ‘from’ and ‘to’ in respective selection boxes.
  • Click on the “Get Results” button in the respective section to obtain of the institutes and programs where you have chances to get a seat.
